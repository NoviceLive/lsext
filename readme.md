# lsext
# 文件格式分布分析工具

## Features
## 描述

- List all unique suffixes
- 可以只列举不同的后缀

- List according to the file count
- 可以按不同后缀的文件数排列显示

- List according to the total file size
- 可以按不同后缀的文件总体积排列显示

## 依赖
